Politica de Privacidad

Skill: Mi Cancionero

Ultima actualizacion: 22 de junio de 2026

Informacion que recopilamos

Mi Cancionero no recopila, almacena ni comparte informacion personal identificable de los usuarios. El unico dato procesado es el termino de busqueda musical que el usuario proporciona mediante comandos de voz (por ejemplo, "pon musica de bad bunny"), el cual se usa exclusivamente para buscar y reproducir el audio solicitado a traves de YouTube Music.

Uso de la informacion

Los terminos de busqueda se envian a YouTube Music unicamente con el proposito de obtener el audio solicitado. No se registran, almacenan ni asocian a ningun perfil de usuario.

Autenticacion

El skill utiliza una cuenta de YouTube Music autenticada para poder acceder al contenido musical. Esta autenticacion es interna del servidor y no requiere ni expone informacion de la cuenta del usuario de Alexa.

Datos de terceros

No compartimos ningun dato con terceros. El unico servicio externo utilizado es YouTube Music, unicamente para la reproduccion de audio solicitada explicitamente por el usuario.

Seguridad

La comunicacion entre el skill de Alexa y el servidor backend esta cifrada mediante TLS. No se almacenan datos de usuario en ningun momento.
